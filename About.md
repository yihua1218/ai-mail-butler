## Why This Project Exists

This project is my practical lab for AI-assisted development.

The goal is simple:
build a new solution end to end with AI tools, then honestly compare the result with traditional development.

I use multiple toolchains and model ecosystems together, including:

- Gemini
- OpenCode
- VS Code
- GitHub Copilot
- OpenAI
- Grok
- OpenRouter
- LM Studio

Cloud and local models, free and paid options, all in one workflow.

They are used across planning, specification writing, docs, coding, frontend work, unit tests, deployment automation, and integration tests.

## What I Am Testing

I want evidence, not hype:

- What each AI tool is truly good at
- Where each tool fails
- What risks appear in real project workflows
- How they differ in speed, quality, and controllability
